🌌 Solar System Explorer
An interactive, responsive, and visually stunning web-based cosmic journey through our solar system. Built using vanilla front-end web technologies, this project showcases an animated solar orrery, a dynamic twinkling starfield, and comprehensive educational profiles for the Sun and all nine planets (including dwarf planet Pluto!).

✨ Features
Interactive Solar Orrery: A real-time, pseudo-3D CSS/JS-animated model of the solar system in the hero section that dynamically tracks orbiting planetary bodies.

Dynamic Canvas Starfield: An optimized HTML5 Canvas backdrop drawing thousands of randomized, softly twinkling stars that adjust flawlessly to screen resizing.

Seamless Navigation: A sticky, blur-effect (backdrop-filter) navigation bar that allows smooth scrolling straight to individual planetary profiles.

Custom CSS-Rendered Planets: Unique visual designs using layered radial gradients, shadows, and inline CSS layout masks to mimic atmospheric features, rings (Saturn), and storm spots (Jupiter).

Scroll-to-Reveal Animation: Clean, modern scroll transitions powered by the IntersectionObserver API to organically slide elements into view.

Curated Fun Facts: A modern grid container layout packed with fascinating space trivia and subtle hover effects.

100% Fully Responsive: Tailored layouts adapting effortlessly from ultra-wide desktops to small smartphones.

🛠️ Tech Stack
This project is built from scratch without any heavy frameworks or dependencies, keeping it incredibly performant, lightweight, and fast.

HTML5: Semantic architecture for layout and cross-device scaling.

CSS3: Custom properties (Variables), Flexbox, CSS Grid, Advanced Keyframe Animations, Perspective transforms, and Media Queries.

JavaScript (ES6+): Vanilla script featuring the Canvas API, modern DOM manipulation, object arrays, and the modern Intersection Observer API.

Typography: Clean, premium type pairings from Google Fonts (Playfair Display & Raleway).
